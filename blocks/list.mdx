The **List** block allows you to present multiple items in a clear, organized way.\
Use it to group related updates, steps, or highlights.

#### Features:

- Supports bullet and numbered lists
- Great for summarizing multiple changes
- Easy to scan and understand

![List Block Dsgn Pn](/images/list_block_dsgn.png)

#### Example use cases:

- Listing new features or bug fixes
- Outlining steps or improvements
- Summarizing minor updates

Lists are perfect for keeping your changelogs concise and user-friendly.
