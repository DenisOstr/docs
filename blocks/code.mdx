The **Code** block lets you share snippets of code or configuration directly in your changelog.\
It's perfect for technical updates or developer-focused releases.

#### Features:

- Syntax highlighting for better readability
- Supports multiple languages
- Great for sharing examples or setup instructions

![Code Block Dsgn Pn](/images/code_block_dsgn.png)

#### Example use cases:

- Showing how to use a new API
- Displaying updated configuration
- Sharing code-level fixes or improvements

Use the Code block to clearly communicate changes to technical users.
