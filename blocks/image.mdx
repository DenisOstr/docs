The **Image** block allows you to add visuals to your changelog posts.\
It’s perfect for showcasing UI changes, diagrams, or any supporting content.

#### Features:

- Upload or embed images
- Full-width or inline display
- Alt text support for accessibility

![Image Block Dsgn Pn](/images/image_block_dsgn.png)

#### Example use cases:

- Showing updated UI components
- Adding visual context to new features
- Including product screenshots in releases

Images help users understand changes faster — a picture says more than a thousand words.
