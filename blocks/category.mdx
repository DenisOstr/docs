The **Category** block lets you group updates under custom labels like _New_, _Fixed_, or _Improved_.\
It helps users quickly understand the type of changes included.

#### Features:

- Customizable category labels
- Visually separates different types of updates
- Helps improve readability and organization

![Category Block Dsgn Pn](/images/category_block_dsgn.png)

#### Example use cases:

- Grouping updates by type: _Bug Fixes_, _Improvements_, _Breaking Changes_
- Creating consistent changelog formatting
- Helping users scan content faster

Use Category blocks to bring structure and clarity to your release notes.
